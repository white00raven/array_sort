Программа в которой реализован класс с сортировками массива и поиском .
Программа позволяет :
1-заполнить массив рандомными числами от1 до 100
2-записать массив
3-отсортировать массив пузырьком
4-отсортировать массив минимумами
5-отсортировать массив вставкой
6-перезаписать массив
7-использовать бинарный поиск
8-использовать поиск линейный
9-отсортировать массив слиянием
10-очистка консоли
11-выход
